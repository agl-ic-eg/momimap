# Description
The momimap is tiny background map viewer for cluster reference GUI.  

# Requirement
These libraries are required.  
- Qt5.12 or later version


# Usage

```sh
export MOMIMAP_MAPBOX_ACCESS_TOKEN=set your access token
/usr/bin/momimap
```
